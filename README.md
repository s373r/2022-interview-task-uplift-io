## Run

```shell
$ npm install
$ npm run start:prod
```
## Local development

```bash
# one-shot start
$ npm run start

# watch mode
$ npm run start:dev
```

Tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
